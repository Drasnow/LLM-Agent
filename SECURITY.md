# Security Policy

这是一个以内容整理为主的资源仓库，不涉及线上服务或发布软件包，但仍然欢迎负责任地报告以下问题：

- 恶意链接
- 钓鱼内容
- 明显高风险的第三方资源
- 泄露隐私或不当公开的联系信息

## 如何报告

请不要优先公开提交安全相关 Issue。

如果你发现上述问题，请通过维护者的私下联系方式联系处理：

- Email: `jiangxin123@users.noreply.github.com`
- GitHub: `@jiangxin123`

## 响应策略

- 维护者会尽快核实问题
- 一经确认，会优先移除恶意链接或高风险内容
- 如果问题涉及第三方平台内容变更，仓库会尽快更新说明或删除该资源

## 范围说明

本仓库的安全策略重点关注内容与链接风险，不处理传统软件漏洞披露流程。
